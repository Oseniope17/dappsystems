What is Dappsystems Pro?

Dappsystems Pro is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making Decentralized wallets a safer choice compared to desktop or browser extension wallets.
